Uniandes Design System

About the project

The University of the Andes is the best private university in Colombia. Its regional presence requires a robust web strategy for more than 500 websites, both informational and transactional. The project was born out of the need to rethink the University's web ecosystem from a different perspective: using UX/UI design methodologies to change this context.

Methodology's

We performed a UI audit to establish what pain points we could find out. This audit resulted in the decision of not only building the Design System, but also to start a redesign.

Benchmark

In the education sector there are many institutions using Design Systems. We performed a thorough analysis of their approach to designing for the web. Almost all the highest ranked educational institutions use a Design System for their information and transaction web sites.

The result

Using the atomic design framework developed by Brad Frost we started the construction of the Design System by establishing design principles that would guide anyone designing or developing for the Design System. In this stage it was crucial to have the participation and point of view of front end developers, back end developers and other key stakeholders.
